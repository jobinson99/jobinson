【2012】我的技术相关名词翻译整理（不完成）

摘要：

话题： 名词 1   翻译 1   中文化 2  

类别： 技术普及 创建时间：2011-02-18

这几年参与开源社区，搞了很多翻译，最近因为我的翻译 LFS为”开天辟地你牛叉“操作系统，而备受关注，也引发了巨大争议，甚至引发freebsd的人对我搞全网追杀，遂决定整理下我近年来名词的翻译成果，并详细解读翻译理由（因为比较分散，整理时间可能会很长，也就是说，此文可能会太监）

一、玩笑版：

Linux相关：

  • 贴身=Tizen
  • 方式：音译+意译
  • 理由：
  • 1、发音接近

  • 2、因为tizen主攻后现代移动办公，属于“贴身”事物，因此用贴身这个名字切合其应用环境。

  • 你牛叉 = Linux
  • 方式：音译
  • 理由：
  • 1、linux如果要发音的话，发音量比你牛叉多，所以你牛叉的翻译比保留英文要省俭。
  • 2、突出自定义性，你不牛叉，怎么能够自定义呢？
  • 3、突出和gnu社区的联系，gnu社区的标志是牛头，那么你牛叉和gnu标志从此可以相互联系起来，构建完整的意义。
  • 4、牛叉，也说明该系统的强大。
  • 5、活泼有趣，心情好点，可让人年轻几岁
  • 不好的地方：

  • 好像有点口语化了，不够装叉，因此不好用来忽悠陌生人。不过“维他命“也很口语化，但并不影响其专业术语的地位。

  • 开天辟地 你牛叉 = LFS(linux from scratch)
  • 方式：意译
  • 理由：
  • 1、名字比较霸气

  • 2、

  • 你牛叉 龙骨 = Arch linux
  • 方式：意译

  • 理由：

  • 你牛叉有奔头 = Ubuntu Linux
  • 方式：音译
  • 有奔头（乌斑兔ubuntu）者，你牛叉(linux)系，蝶变(debian)之化物，期以仁者待人之许，赠卡(cd)于乐之者（geeker），广收众心，益于其助之智器(computer)，童叟皆可擅控。
  • 理由：
  • 1、表达良好前景和期望

  • 2、

  • 蝶变你牛叉 = Debian Linux
  • 方式：意译+音译
  • 理由：
  • 1、发音接近
  • 2、debian是夫妇合名，是爱情结晶，而中国恰好有个梁山伯祝英台的化蝶爱情故事。
  • 3、意义上契合本身特色：一代代稳定地演化，蝶变的软件包测试流程确是一“蝶变”的过程

  • 4、此名字也是希望蝶变能够像“蝴蝶效应”一样，持续传播自己的影响力。

  • 合德 ＝ Hurd
  • 方式：音译
  • 理由：
  • 1、体现gnu的同心同德精神

  • 2、“合德”中文发音近“核的”，恰如其分体现合德本身作为内核的重要作用。

  • 共努＝ GNU
  • 方式：音译＋音译
  • 理由：
  • 1、gnu社区是个共同努力维护的社区

Arch linux相关：

  • 打包党 = pacman
  • 方式：意译
  • 理由：
  • 1、你牛叉发行版的特色很多因素来自软件包管理方式，其中，打包方式是重点因素之一，这个是一般你牛叉发行版的核心动力。

  • 2、很多人有个感觉，就是所谓发行版不过就是打包党，因此，作为软件管理工具，pacman化身为打包党是合理的。

  • 图基系统 = X windows system
  • 方式：意译
  • 1、连接系统核心和顶层窗口管理界面的中间层。
  • 2、这层是抽象的，也是基础的，不会有太多的效果，但其提供的基本效果组合，可以组成顶层界面的所有效果

Lisp相关

  • 链码 ＝ Lisp
  • 方式：音译、意译
  • 理由：
  • 1、lisp语言是语法树非常清晰直观的语言，像一条条链。

  • 2、

  • 通链 ＝ common lisp

  • 中国链 ＝ China Lisp User Group
  • 方式：意译

  • 理由;1、全称“中国链码用户组”的简化昵称

  • 易码肆 = Emacs
  • 方式：音译、意译
  • 理由：
  • 1、emacs确实提高了编码效率，所以称易码
  • 2、发音接近。

  • 3、肆，古代表示手工作坊，此工具确实是一个电脑版的手工作坊工具集

  • 名位 = namespace
  • 方式：音译
  • 理由：

  • 1、namespace的意义在于追求精确 定 “位”，属于名－位相互精确一一对应，因此译为名位，而不是命名空间这种生硬直译。

  • 串 ＝ List
  • 方式：意译
  • 理由：
  • 1、list是链码的核心概念，因此必须正确把握好其内涵和外延，方能理解学习链码这种编程语言，而串是个象形字，恰好形象表现了list单维度属性。

  • 2、list在链码中可以表达：数、数处理符、字符、数组、多维数组、对象等，因此，不是“表”这个字的内涵所能包括的。

  • 守宫＝gecko（火狐的核心）
  • 方式：意译

  • 理由：

  • 世通码／万国码＝ unicode
  • 方式：意译

  • 理由：

  • 囧境 ＝ jail （freebsd）
  • 方式：意译，形译
  • 理由：
  • 1、freebsd一个重要特性，作为隔离的虚拟环境，确保软件不会影响基本系统。jail是监狱，但如果直接译成监狱或者囹圄会让名词太长，直接短译为囧境＝囧虚拟隔离环境。

  • 2、囧在这里是象形，隔离的意思，呵呵，中文象形优势尽显。

  • 虚存＝ swap
  • 方式：意译

  • 理由：虚拟内存的简称，对应的是硬存＝内存条。

  • 芯存=RAM
  • 方式：意译
  • 理由：芯是指核心芯片。

版本管理相关

  • 提案 = stage

  • 方式：意译

  • 立案 = commit
  • 方式：意译
  • 理由：

  • 1、需要有权限的，才能提交，因此是正式的，用提案较好

  • 贡献者／志愿者／委员 = committer
  • 方式：意译
  • 理由：
  • 1、需要一定门槛，也有一定的权力
  • 2、注意：翻译为委员仅限freebsd，开放性的组织，这个应该翻译为贡献者，志愿者
